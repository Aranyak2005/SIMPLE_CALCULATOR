#  Simple Calculator

A clean, simple, and responsive calculator built using **HTML, CSS, and JavaScript**.  
This project focuses on minimal design, smooth user experience, and core JavaScript logic — no APIs used.

##  Features
- Basic arithmetic operations (+, −, ×, ÷)
- Percentage (%) calculation
- Clear (C) and Backspace (⌫) functionality
- Responsive design (mobile & desktop friendly)
- Clean and modern UI with gradient background
- Lightweight and fast

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)


##  How It Works
- User clicks buttons to input values
- JavaScript captures button clicks
- Expressions are evaluated using built-in JS logic
- Output is displayed dynamically on screen



##  How to Run Locally
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Start calculating
